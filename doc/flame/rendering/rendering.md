# Рендеринг

Рендеринг — это то, как ваша игра отрисовывает всё, что видит игрок: спрайты, текст, эффекты частиц и пользовательские фигуры. Flame надстраивается над API [Canvas](https://api.flutter.dev/flutter/dart-ui/Canvas-class.html) Flutter и добавляет игровые утилиты для загрузки спрайт-листов, анимации кадров, рендеринга текста с растровыми шрифтами, применения визуальных декораторов и выполнения постобработки с помощью шейдеров.
